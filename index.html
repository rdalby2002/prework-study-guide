HTML

  • The head element (AKA tag) contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used to determine the structure and content of a webpage.
  • Info stored in the head is known as "metadata," which is a set of elements invisible to users that acts as the base for the content included in the body.
  • The body denotes a webpage's visible content, such as text, images, and hyperlinks.
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets (< >).
  • The child elements of the body are "semantic," meaning the name of the element describes its content and the role it plays on a webpage.
  • When possible, it is important to use semantic HTML elements, because they make your HTML easier to read by other developers and the browser.
  • The !DOCTYPE html declaration informs the browser of the document type.
  • The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • The style element defines the CSS styles associated with a website.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The script element embeds client-side scripts in the HTML document.
  • The most commonly used charset is UTF-8, which allows nearly all characters to be used and displayed in a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A link element must be established in the header of your HTML code to apply an external style sheet to your webpage.
  • CSS rules consist of a selector and a set of declarations.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property
  • A selector can be applied to all elements within a certain class of your webpage by placing a period before your selector.
  • The border property assigns a border around your element.
  • The box-shadow CSS property adds shadow effects around an element's frame.
  • The width property attributes the percentage of available width that your element will occupy.
  • When we want to apply a rule to all the elements visible on the page, we use a special wild card symbol, or *.
  • Two selectors cam be combined into a single CSS rule with a comma, allowing declarations to affect both selectors.
  • The positioning property text-align can be used to center a title or heading on a webpage.
  • The img selector assigns property values to any image element in the markup.
  • To make each image on a webpage start on a new line, assign the block value to its disply property.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url]: retrieve an entire repository from a hosted location via URL
  • git add [file]: add a file as it looks now to your next commit (stage)
  • git push [alias]/[branch]: Transmit local branch commits to the remote repository branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • querySelector is used to grab a reference to an HTML element that can then be stored in a variable.
  • A semicolon at the end of a line indicates where a statement ends. It is only required when you need to separate statements on a single line.
  • JavaScript is case sensitive.
  • After declaring a variable, you can give it a value using =
  • You retrieve the value by calling the variable name.
  • After assigning a value to a variable, you can change it later in the code.
  • An operator is a mathematical symbol that produces a result based on two values (or variables).
  • We can use the console.log method to output a message to the web console by adding an argument.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
